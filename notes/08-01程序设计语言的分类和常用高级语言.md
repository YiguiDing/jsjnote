# 08-01程序设计语言的分类和常用高级语言
## 程序设计语言的分类
1. 机器语言
1. 汇编语言
1. 高级语言

>## 翻译程序的分类
>1. 汇编程序
>    - 汇编源代码->可执行目标程序
>1. 解释程序
>    - 高级程序源代码->由解释器解释执行
>1. 编译程序
>    - 高级语言源代码 -> 可执行目标程序

## 常用高级语言
> c c++
> java
> python
