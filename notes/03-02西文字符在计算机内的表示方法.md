# 03-02西文字符在计算机内的表示方法
* 两种重要西文字符编码方式：**ASCII码**和**EBCDIC码** 
* 目前计算机中普遍采用的是ASCII（American Standard Code for Information Interchange）码，即美国信息交换标准代码
* ASCII码有7位版本和8位版本两种，国际上通用的是7位版本，7位版本的ASCII码有128个元素
    >0000 0000 ~ 0111 1111 0~127共128个数
    ASCII推算
    '0'->48
    'A'->65
    'a'->97
    'a'-'A' = 23
